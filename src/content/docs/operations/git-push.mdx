# GitHub SOP

## How to push to GitHub to trigger a build on Netlify

```bash
# Navigate to your project directory (replace 'your_project_directory' with your actual directory path)
cd your_project_directory

# Check the status of your Git repository
git status

# Add all the changes you want to commit (replace 'your_commit_message' with your actual message)
git add .

# Commit your changes
git commit -m "your_commit_message"

# Push your changes to the remote repository named "origin"
git push origin master
```
